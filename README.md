## Run Locally

Clone the project

```bash
  git clone https://github.com/Preetibildani/average-pig-3056.git
```

Go to the project directory

```bash
  cd Expedia
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
